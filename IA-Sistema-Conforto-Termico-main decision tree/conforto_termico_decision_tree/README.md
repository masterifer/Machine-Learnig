# Sistema de Conforto Térmico com Decision Tree

## Como executar

1. Abra a pasta no VS Code
2. Abra o terminal
3. Instale as bibliotecas:

pip install -r requirements.txt

4. Execute:

python main.py

## Objetivo

Classificar conforto térmico usando:
- Regras if/else
- Machine Learning com Decision Tree
